# 🌐 Sahid Alam - Personal Portfolio

A modern, responsive, and full-stack personal portfolio website showcasing my skills, projects, and experience as a Full Stack Developer.

---

## 🔥 Live Demo
👉 https://personal-portfolio-web-woad.vercel.app

---

## 📌 Features

- ✨ Modern UI with smooth animations (Framer Motion)
- 📱 Fully responsive design (Mobile + Desktop)
- 📂 Projects showcase with live demo & GitHub links
- 📬 Contact form with backend integration
- ☁️ Data stored in MongoDB Atlas
- ⚡ Fast performance & optimized loading
- 🔒 Secure API handling

---

## 🛠️ Tech Stack

### 🎨 Frontend
- React.js
- Tailwind CSS
- Framer Motion

### ⚙️ Backend
- Node.js
- Express.js

### 🗄️ Database
- MongoDB Atlas

### 🚀 Deployment
- Vercel (Frontend)
- Render (Backend)
